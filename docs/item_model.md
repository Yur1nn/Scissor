# ItemModel System

## Overview
The ItemModel system allows you to define and register custom items, weapons, armor, tools, furniture, and more, directly in the core server. Each item can have unique properties, types, and metadata.

## Features
- Unique ID and display name for each item
- Item types: Weapon, Armor, Tool, Furniture, Misc
- Extensible properties (damage, durability, custom data, etc.)
- Central registry for all custom items

## Usage
- Define new items in code or via config (planned)
- Register items with the ItemModelRegistry
- Retrieve and use custom items in gameplay logic

## Expansion
- Add support for textures, models, and lore
- Allow loading items from configuration files
- Integrate with plugin API for dynamic content

Update this file as the system evolves.
