# Features

## Implemented
- Loads configuration from config.toml (TOML format)
- Loads static schematic-based maps (.schematic, NBT format)
- Core world manager supports multiple worlds
- Block and semi-block (block entity) structure (semi-block parsing stub)
- ItemModel system for custom items, weapons, armor, tools, furniture, etc.
- World control methods (time, PvP, block permissions, etc.)
- Project is git-ready with CI and PR template

## Planned
- Schematic block ID to name/property mapping
- Semi-block (block entity) parsing and restoration
- Plugin API and dynamic loading
- Networking and protocol support
- Admin commands and server loop
- Map conversion tools

Update this file as features are added or changed.
