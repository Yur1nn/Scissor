# PvP Mechanics

## Overview
- The server supports switching between 1.8 (legacy) and modern PvP mechanics.
- Near-vanilla replication is a goal for both modes.

## Modes
- **Legacy 1.8:** Fast attack speed, no cooldown, classic knockback.
- **Modern:** Attack cooldown, shield support, new knockback.

## Switching
- PvP mode can be set per-arena or globally (planned).
- Exposed via config and API.

## Implementation Notes
- Keep PvP logic modular for easy updates.
- Document any protocol or mechanic differences.
